# trails-example-aurelia based on demo-materialize 
1. git clone https://github.com/johntom/trails-example-aurelia.git
2. cd trails-example-aurelia
3. npm install
4. npm install jspm
5. .\node_modules\.bin\jspm install
6. gulp watch
7. or gulp serve (with bundle)
